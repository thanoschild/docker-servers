pm2 commands:
-> check running app
pm2 list
-> stop an app
m2 stop <app_name_or_id>
-> restart an app
pm2 restart <app_name_or_id>
-> stop and delete an app
pm2 delete <app_name_or_id>
-> check logs
pm2 logs <app_name_or_id>
-> save process list
pm2 save
-> set pm2 to auto restart on reboot server
pm2 startup
